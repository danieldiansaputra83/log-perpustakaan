# log-perpustakaan
Library Management System adalah aplikasi manajemen perpustakaan yang berfokus pada efisiensi pengelolaan data buku, anggota, dan transaksi peminjaman. Proyek ini mendemonstrasikan implementasi basis data relasional yang teroptimasi melalui perancangan ERD (Entity Relationship Diagram) yang komprehensif dan manajemen kueri SQL.
